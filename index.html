Usage

1. Include the listy-text.css and listy-text.js files on your page (duh!):

<script src="listy-text.js" type="text/javascript"></script>
<link rel="stylesheet" type="text/css" href="listy-text.css">

2. Add an input element and call the .listy() function on it:

<input class="listy-text" type="text" />
<script>
$(".listy-text").listy();
</script>
  • Note that the class of the input doesn't have to be 'listy-text'. You can call the .listy() function using any css selector.
  • To use one of the included themes shown above, simply add the bootstrappy or facebooky class to the input element.

3. Get the listy-data:

Use the .listyVal() function to get the data for a particular listy-input. Use css selectors to specify.

<script>
$("form input.listy-text").listyVal();
</script>
